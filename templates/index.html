About the Project

Quote Sentiment Analysis is a sophisticated web application designed to provide insights into the sentiment of various quotes. Using advanced AI algorithms from Azure Text Analytics, the application evaluates the sentiment of a given quote and provides a concise summary.

How to Use

Click the "New Quote" button below to fetch a new quote. You'll see the sentiment and a summary of the quote displayed on the page.

Backend and Technologies

The backend of this application is built using Flask, a lightweight Python web framework. It connects to an

Azure Database for MySQL

to fetch quotes and uses

Azure Text Analytics

for sentiment analysis.

Technologies Used: Python, Flask, Azure Text Analytics, MySQL, HTML, CSS, JavaScript.
